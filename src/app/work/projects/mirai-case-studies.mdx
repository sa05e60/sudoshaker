## Overview

Mirai is a cost-conscious adaptive tutoring and artifact-generation app built to help learners move from conversation to structured outputs without wasteful generation loops. The product combines chat memory, artifact planning, and deterministic export flows into a focused learning workspace.

## Product Summary

Mirai currently focuses on:

- adaptive chat with stored behavioral and linguistic signals
- artifact generation from chat context
- artifact refinement and version history
- deterministic export flows
- conservative usage controls to protect cost
- responsive chat and artifact UX across desktop and mobile

## Scope Decisions

- notebooks and notebook-driven study flows are temporarily removed from the active surface
- exports are deterministic only
- artifact generation is intentionally conservative to control spend
- refinement is bounded and revision-capped

## Tech Stack

- FastAPI
- SQLAlchemy
- Alembic
- vanilla TypeScript frontend

## Outcome

The result is a lean tutoring system that keeps the product affordable while still supporting adaptive teaching, artifact creation, and a clean user experience.
